💻 コマンド例

マインクラフト
コマンド例

実際に使えるコマンドを体験してみよう！
クリックするとコピーできます。

基本コマンド

// テレポート
/tp @s 0 64 0
クリックでコピー
// アイテムを出す
/give @s diamond_sword 1
クリックでコピー
// ブロックを設置
/setblock ~ ~-1 ~ glass
クリックでコピー
// 時間を変更
/time set day
クリックでコピー

応用コマンド

// 範囲にブロック設置
/fill ~-5 ~-1 ~-5 ~5 ~-1 ~5 glass
クリックでコピー
// エフェクトを与える
/effect give @s speed 30 2
クリックでコピー
// エンティティを召喚
/summon zombie ~ ~ ~
クリックでコピー
// 全員にアイテム
/give @a diamond 10
クリックでコピー
コマンドリスト完全版をダウンロード
